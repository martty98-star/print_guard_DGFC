# PrintGuard Core

## Role
PrintGuard is an offline-first browser application plus Netlify-backed APIs for print operations, order tracking, stock handling, checklist workflows, and production reporting. `app.js` still acts as the main orchestrator while more domain logic is being split into modules under `app/` and `scripts/`.
