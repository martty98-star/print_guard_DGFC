Runtime shape
03

Frontend

Classic-script vanilla JS PWA. Shell je v index.html, orchestraci stále částečně drží app.js, nové části jsou v app/, scripts/ a reports/.

Backend

Netlify Functions pro HTTP API. Sdílené helpery jsou v netlify/functions/_lib/.

Database

Neon PostgreSQL. Persistují se sklad, Colorado záznamy, print accounting, processed ordery, push subscriptions a checklist audit.

Server pipelines

Windows/Node/PowerShell skripty pro NAS, Submit Tool, Post Purchase a Colorado accounting.
